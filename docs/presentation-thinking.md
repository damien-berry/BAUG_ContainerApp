# Presentation thinking

This document captures the core idea for the Brisbane Azure User Group talk:

- Azure Container Apps as the runtime for production AI agents
- Useful pattern or build demo hype?
- Focus on real operational value, not just novelty

Suggested discussion points:

1. What makes Container Apps a sensible runtime for agent workloads?
2. Where the pattern helps in production scenarios.
3. What risks or limitations should be called out clearly.
4. How to keep the demo grounded in practical engineering outcomes.
